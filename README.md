# Logitravel Frontend Challenge

Aplicación para gestionar una lista de cadenas de texto. Implementada en dos versiones: Vanilla JS y React.

## Descripción

La app permite añadir y eliminar entradas de texto de una lista, con validaciones y mejoras de UX.

## Funcionalidades

### Requeridas

- [ ] Contenedor donde se muestran las cadenas de texto
- [ ] Input de texto para escribir nuevas entradas
- [ ] Botón para agregar una entrada a la lista
- [ ] Botón para eliminar la entrada seleccionada
- [ ] No se pueden añadir entradas vacías
- [ ] Los ítems son seleccionables; no se puede eliminar sin seleccionar antes
- [ ] Selección múltiple y borrado de varios ítems a la vez
- [ ] Eliminar ítem con doble click
- [ ] Botón de deshacer (undo) el último cambio
